Restaurant Menu & Order Management System

Overview
This project implements a Restaurant Order Management System using Python core data structures such as lists, dictionaries, and nested structures.

The project is divided into four tasks.

------------------------------------------------

Task 1 – Menu Exploration

This task analyzes the restaurant menu stored as a dictionary.

Features:
• Displays the menu grouped by category
• Shows item price and availability
• Calculates statistics such as:
  - Total menu items
  - Available items
  - Most expensive item
  - Items priced below ₹150

------------------------------------------------

Task 2 – Cart Operations

This task simulates a customer placing an order.

Features:
• Add items to the cart
• Remove items from the cart
• Update item quantities
• Prevent adding unavailable or non-existent items
• Display order summary with GST calculation


------------------------------------------------

Task 3 – Inventory Tracker

This task manages kitchen inventory.

Features:
• Creates a deep copy backup of the inventory
• Demonstrates deep copy protection
• Deducts stock based on customer orders
• Prevents stock from going below zero
• Displays reorder alerts when stock reaches reorder level


------------------------------------------------

Task 4 – Sales Log Analysis

This task analyzes historical sales data.

Features:
• Calculates total revenue per day
• Finds the best-selling day
• Identifies the most ordered menu item
• Adds a new day's sales data
• Displays a numbered list of all orders

------------------------------------------------
